Installation
============

Dependency
-----------------

EyeChecker(0.9.14+)依赖 libcurl4 需要安装

.. code:: bash

    sudo apt install libcurl4

`yum` 方式同理, 没有用到对应的组件可以不安装

Install from pypi
-----------------

.. code:: bash

   pip install nlpir-python

Install from source
-------------------

.. code:: bash

   python3 setup.py install


Run test with pytest
--------------------

.. code:: bash

    pytest -vv
